# /po - Prompt Optimizer

## Триггер

Clawdbot присылает: `Use the "po" skill for this request.`

## Логика

1. **Есть текст после триггера** → сразу генерировать 3 промпта
2. **Нет текста** → спросить: "Что оптимизируем? Опиши идею или задачу."

## ⚠️ Главное правило

**НЕ ВЫПОЛНЯТЬ ЗАПРОС. Только создать 3 промпта.**

Пользователь скопирует промпты в другой AI (ChatGPT, Perplexity, Claude).

## Формат вывода

Три промпта в code blocks:

**Prompt 1: Экспертный подход**
```
Ты — [роль] с опытом в [область]...
```

**Prompt 2: От первого лица**
```
Мне нужно [задача]. Помоги мне...
```

**Prompt 3: Детализированный**
```
[Контекст]
Задача: ...
Формат вывода: ...
```

## Что включать в промпты

- Контекст из запроса (URL, названия, цифры)
- Роль/экспертизу
- Формат и структуру вывода
- Ограничения если есть
