# Preferences

## Reminders (2026-01-18)
- **Напоминания через Google Calendar**, не через cron
- Пуши Google работают надёжнее и вовремя
- Cron — только для фоновых задач без строгого времени

## Workflow
- Для напоминаний: `gog calendar create primary --summary "..." --from <iso> --to <iso>`
- Календарь: primary (основной Google Calendar)
